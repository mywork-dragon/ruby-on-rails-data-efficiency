    Introduction

    Welcome to the MightySignal API!

    The MightySignal API lets you interact with MightySignal from anything that can send an HTTP request.

    Using the API, you can access MightySignal’s unique data on millions of mobile apps.

    Authentication

    MightySignal uses API keys to allows access to the API. Contact MightySignal to get your API key.

    The API expects for the API key to be included in all API requests to the server in a header that looks like the following:

    MightySignal-API-Key: YourApiKey

    You must replace YourApiKey with your own API key.

    This endpoint retrieves an iOS app.

    HTTP Request

    GET http://api.mightysignal.com/ios_apps?appStoreId=<appStoreId>

    URL Parameters

    Parameter Description
    appStoreId The ID of the iOS app to retrieve. For example, Instagram’s ID is 389801252. An easy way to get an app’s ID is to extract it from its App Store URL; that is, extract 389801252 from https://itunes.apple.com/us/app/instagram/id389801252.

    Response

    Key Description
    mightySignalId MightySignal’s unique ID for the app
    name The name of the app
    mobilePriority How much the company cares about the app. Can be high, medium, or low. Use this to filter out apps that are not actively being developed or marketed. Currently, the rank is a function of how recently their app has been updated, and whether they advertise on Facebook.
    adSpend Whether the app is advertising on Facebook
    userBase A tier representing how many active users the app has. Can be elite, strong, moderate, or weak.
    appStoreId The app’s unique identifier on the App Store. It’s the same value that is passed in as a parameter.
    iconLarge Link to the app’s large icon
    iconSmall Link to the app’s small icon
    company See COMPANY table

    Company

    Key Description
    mightySignalId MightySignal’s unique ID for the company
    name The name of the company
    fortuneRank The Fortune rank
    streetAddress The address of the company
    city The city where the company is located
    zipCode The zip code where the company is located
    state The state where the company is located

    Errors

    The MightySignal API uses the following error codes:

    Error Code Meaning
    400 Bad Request
    401 Unauthorized – Your API key is wrong.
    503 Service Unavailable – We’re temporarily offline for maintanance. Please try again later.
