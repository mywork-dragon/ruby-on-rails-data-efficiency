    Introduction

    Welcome to the MightySignal API!

    Authentication

    MightySignal uses API keys to allows access to the API. Contact MightySignal to get your API key.

    The API expects for the API key to be included in all API requests to the server in a header that looks like the following:

    MightySignal-API-Key: YourApiKey

    You must replace YourApiKey with your own API key.

    Apps

    Get a Specific iOS App

    GET http://api.mightysignal.com/ios_apps?appStoreId=389801252 HTTP/1.1
MightySignal-API-Key: YourApiKey

    This endpoint retrieves a iOS app.

    HTTP Request

    GET http://api.mightysignal.com/ios_apps?appStoreId=<appStoreId>

    URL Parameters

    Parameter Description
    appStoreIddd The ID of the iOS app to retrieve. For example, Instagram’s ID is 389801252. An easy way to get an app’s ID is to extract it from its App Store URL; that is, extract 389801252 from https://itunes.apple.com/us/app/instagram/id389801252.

    Response

    Key Description
    mightySignalId MightySignal’s unique ID for the app

    Get a Specific Android App

    GET http://api.mightysignal.com/android_apps?googlePlayId=com.instagram.android HTTP/1.1
MightySignal-API-Key: YourApiKey

    The above command returns JSON structured like this:

    {
  "id": 2,
  "name": "Isis",
  "breed": "unknown",
  "fluffiness": 5,
  "cuteness": 10
}

    This endpoint retrieves an Android app.

    HTTP Request

    GET http://api.mightysignal.com/android_apps?googlePlayId=<googlePlayId>

    URL Parameters

    Parameter Description
    googlePlayId The ID of the Android app to retrieve. For example, Instagram’s ID is com.instagram.android. An easy way to get an app’s ID is to extract it from its Google Play URL; that is, extract com.instagram.android from https://play.google.com/store/apps/details?id=com.instagram.android.

    Companies

    Get a Company and Its Apps (Given a Website)

    GET http://api.mightysignal.com/companies?website=instagram.com HTTP/1.1
MightySignal-API-Key: YourApiKey

    The above command returns JSON structured like this:

    {
  "id": 2,
  "name": "Isis",
  "breed": "unknown",
  "fluffiness": 5,
  "cuteness": 10
}

    This endpoint retrieves a company and its information plus all of its apps and their information.

    HTTP Request

    GET http://api.mightysignal.com/companies?website=<website>

    URL Parameters

    Parameter Description
    website The website that you want to search. Use the domain only without http or www in front of it. For example, use instagram.com instead of http://instagram.com.

    Errors

    The MightySignal API uses the following error codes:

    Error Code Meaning
    400 Bad Request
    401 Unauthorized – Your API key is wrong.
    503 Service Unavailable – We’re temporarially offline for maintanance. Please try again later.
